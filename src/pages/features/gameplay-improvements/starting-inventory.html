You can make players always spawn with inventory items by enabling the flags at the menu 
Netplay Options > Starting Inventory
. Você pode fazer os jogadores sempre nascerem com determinadas items no inventário habilitando as flags no menu 
Netplay Options > Starting Inventory
. The number shown above the menu is a bitfield representing the selected inventory items, go to command line arguments to learn more. You can use the args builder to generate the argument. O número exibido acima do menu é um bitfield que representa os items do inventário selecionados, vá para argumentos de linha de comando para entender mais. Você pode utilizar o construtor de argumentos para gerar o argumento.
