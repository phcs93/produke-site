For team based modes, a team picker is shown at the terminal. After each player has selected a team, the game will start. The host needs to use the argument 
/teampicker
. Para modos baseados em times, um seletor de times é exibido no terminal. Depois que cada jogador selecionar um time, o jogo é iniciado. O host precisa utilizar o argumento 
/teampicker
.
