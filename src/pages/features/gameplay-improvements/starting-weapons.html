You can make players always spawn with certain weapons by enabling the flags at the menu 
Netplay Options > Starting Weapon
. Você pode fazer os jogadores sempre nascerem com determinadas armas habilitando as flags no menu 
Netplay Options > Starting Weapon
. The number shown above the menu is a bitfield representing the selected weapons, go to command line arguments to learn more. You can use the args builder to generate the argument. O número exibido acima do menu é um bitfield que representa as armas selecionadas, vá para argumentos de linha de comando para entender mais. Você pode utilizar o construtor de argumentos para gerar o argumento.
