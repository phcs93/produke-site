By default, and unlike the original game, proDuke will not load custom files (ART, DAT, CON, etc) placed in the game's directory. To disable this, you have to run the game with the custom argument 
/allowmods
. Everyone in the match will be prompted at the console to accept playing with this argument enabled. Por padrão, e diferentemente do jogo original, o proDuke não vai carregar arquivos customizados (ART, DAT, CON, etc) colocados no diretório do jogo. Para desabilitar esta trava, você deverá iniciar o jogo com o argumento 
/allowmods
. Todos na partida deverão aceitar a votação no console para que o jogo inicie com a trava desligada.
